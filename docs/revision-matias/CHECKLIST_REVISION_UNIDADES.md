# Checklist de revisión para Matías

Usar esta lista para revisar cada unidad.

## Contenido técnico

- ¿El contenido base del Documento Madre aparece suficientemente?
- ¿Falta alguna explicación importante?
- ¿Hay alguna afirmación técnica que convenga corregir?
- ¿Hay ejemplos que deban adaptarse al contexto real de espectáculos?
- ¿Hay términos que necesitan glosario?

## Imágenes

- ¿Las imágenes correctas del Documento Madre están presentes?
- ¿Alguna imagen debe cambiarse de unidad?
- ¿Alguna imagen necesita explicación/caption?
- ¿Alguna imagen es confusa para alumno inicial?

## Pedagogía

- ¿La unidad empieza por lo básico?
- ¿El nivel sube progresivamente?
- ¿La actividad obliga a aplicar criterio?
- ¿Las preguntas son demasiado obvias?
- ¿Los distractores son técnicamente creíbles?

## Producto

- ¿La navegación se entiende?
- ¿Modo Alumno sirve para cursado real?
- ¿Modo Dev sirve para revisar rápido?
- ¿La unidad queda demasiado larga o todavía escueta?
- ¿Qué debería tener antes de vender o publicar?

## Evaluación final

- ¿Debe ser multiple choice, casos prácticos o mixta?
- ¿Qué errores graves deberían desaprobar?
- ¿Qué competencias mínimas debe demostrar el alumno?
