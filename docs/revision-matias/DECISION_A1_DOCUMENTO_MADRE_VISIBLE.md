# Decisión A1 — Documento Madre visible

## Decisión

Se mantiene el Documento Madre visible dentro de cada unidad, incluyendo imágenes originales.

## Justificación

Matías indicó que el contenido actual de algunas unidades quedó demasiado escueto y que el Documento Madre debe aparecer. La función del Formato C no es resumir hasta perder la fuente, sino convertirla en una experiencia de aprendizaje clara, navegable y evaluable.

## Estructura editorial por unidad

1. Documento Madre visible
   - texto base;
   - imágenes originales;
   - ubicación de clase/página;
   - notas de lectura cuando haga falta.

2. Desarrollo Blacksmith
   - explicación ampliada;
   - orden pedagógico;
   - corrección suave de imprecisiones;
   - ejemplos de espectáculo, sala, vivo y operación.

3. Refuerzo externo seguro
   - fuentes abiertas, oficiales o institucionales;
   - sitio, autor/organización y condición mínima de uso;
   - sin copiar más de lo necesario.

4. Síntesis pedagógica
   - ideas clave;
   - errores frecuentes;
   - glosario.

5. Aplicación
   - actividad;
   - microdesafío;
   - cuestionario con dificultad progresiva.

## Criterio legal-operativo

Para esta etapa se asume autorización/conformidad suficiente para avanzar con el uso visible del Documento Madre y sus imágenes. Si en el futuro aparece una restricción concreta, se reemplazarán las imágenes necesarias por diagramas propios sin cambiar la estructura pedagógica.
